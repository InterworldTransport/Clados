Clados package(s):

  1. cladosF
  2. cladosFExceptions
  3. cladosG
  4. cladosGExceptions

The clados packages form a working library for applications that would use geometric or clifford algebras to replace vector and matrix libraries. The classes are designed to enable the software developer to work at the geometric level with their designs and let the library handle the underlying algebra and arithmetic. Some rigor is required to use the library effectively, but examples of its use are developed elsewhere.
